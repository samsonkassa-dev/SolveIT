Add a Project

Go to the navigation bar and tap on My Projects. If you can't access the side menu, either you're using a smaller screen device or you have minimized it. In that case, click to toggle the side menu.

By tapping on the My Projects menu option, you'll navigate to Project list page.Tap on . You will be shown a dialog with two sections that have to be filled to create your project, which contains two sections :

Project Details Section
This section contains input for all the basic information about the project . The project name, description ...
Project Proposal Section (not required)
In this section you can upload your project proposal. It is very important that it is detailed as possible so that your mentors and trainers can understand your project provide support accordingly.

Judging Criteria

This judging criteria will explain what to do on the contest and clearly describe the criteria of the judging process. So that participants know how their project will be judged. It will also tell the judging body what to look for and how to assign value or rank. The main subject or subject area is clearly presented by thoughtful placement in the frame. The Jury Members may assign a score of 1-5 based on how well the submission meets the criteria listed on the judging criteria document. Participants are encouraged to meet as many of these criteria as possible, to improve their work and chances of winning. However, it is compulsory to meet these criteria.
